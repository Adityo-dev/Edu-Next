## 📌 সংক্ষিপ্ত পরিচিতি

**EduNext** হলো বাংলাদেশের শিক্ষার্থী ও পেশাদার ইনস্ট্রাক্টরদের জন্য তৈরি একটি ফুল-ফিচার্ড অনলাইন লার্নিং প্ল্যাটফর্ম। এখানে ভেরিফাইড ইনস্ট্রাক্টরদের কাছ থেকে কোর্স কিনে শেখা, লাইভ ক্লাসে যোগ দেওয়া এবং সার্টিফিকেট অর্জন করা যায়। ইনস্ট্রাক্টররাও তাদের জ্ঞান শেয়ার করে আয় করতে পারেন।
